How It Works? The Laundry Metric Ecosystem

1. Smart Sensors

IoT devices (Raspberry Pi with weight sensors) monitor inventory in real-time.

2. Connectivity & Cloud

Data sent to our API in Azure Kubernetes and stored in SQL Server Serverless.

3. Management System

Intuitive web application (Angular 16) to configure and manage your operation.

4. Smart Dashboards

Embedded Power BI dashboards for real-time analysis and decision-making.
